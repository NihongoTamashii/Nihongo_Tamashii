# **App Name**: Kotoba Drill

## Core Features:

- Kotoba Flashcards: Interactive flashcards displaying Japanese vocabulary on one side (hiragana, katakana, or kanji) and the meaning on the reverse side.
- Kotoba Practice: Exercises where the user is prompted with the Indonesian/meaning of a Kotoba and types the Japanese equivalent using the on-screen Japanese keyboard. If an incorrect answer is entered the LLM will act as a tool to check for minor grammatical/spelling errors. 
- Japanese Keyboard: A virtual Japanese keyboard for inputting answers in the Kotoba Practice section.

## Style Guidelines:

- Primary color: Soft blue (#7BB4E5) evoking calm focus for language learning.
- Background color: Light gray (#F0F2F5) provides a neutral backdrop to emphasize content.
- Accent color: Pale orange (#E5987B) draws attention to interactive elements and feedback messages.
- Headline font: 'Poppins' sans-serif font for clarity, especially with Japanese characters.
- Body font: 'PT Sans' for a balance of modern readability and approachability.
- Use simple, clean icons for navigation and function buttons.
- Subtle card flip animations for flashcards and smooth transitions for keyboard appearance.
